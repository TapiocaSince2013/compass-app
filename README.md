# Compass

**いまの単価、安すぎない…?** と思ったこと、ありませんか。

それを確かめられる場所が、これまでありませんでした。
相場の情報は、お店と求人サイトの側にあって、はたらく側にはなかったから。

Compassは、吉原のソープではたらく人のための、無料のツールです。
総額と時間を入れるだけで——

- はたらく人のなかでの、**あなたの位置**(上位何%か)
- あなたの条件からみた目安と、いまの単価の**ズレ**。「**過小評価されているかもしれません**」が分かれば、交渉の材料になります
- お客さまの支払いのうち、**あなたに届いている割合**(取り分の確認)

👉 **使ってみる**: ★URL(あとで私が確定します)★
インストール不要。ブラウザで開くだけ。登録もログインもありません。

出てくる数字は目安で、**あなたの価値の優劣を意味しません**。
上げることだけが正解でもありません。決めるのは、あなたです。

## まもられていること

- 入力は、**この端末から出ません**。送信機能がそもそもありません
- **保存もしません**。画面を閉じれば消えます
- 見られそうなときは、**そっと隠せます**(右上のボタンか、Escキー)

「信じて」とは言いません。このページの上にある index.html が中身のすべてです。
コードが読める人なら、**送信ゼロを自分の目で確かめられます**。

## データ提供について(任意)

いま見えている目安は、作成者が置いた仮のものです。
使った人が匿名データを分けてくれると、目安がほんものの実態に近づいていきます。

- 提供しなくても、ぜんぶの機能が使えます
- 送る内容は、送る前に**全文見えます**。2,500円きざみに丸めた幅と、答えた項目だけ。取り分は**希望したときだけ**(5%きざみに丸めます)。
  正確な金額・店名・名前・連絡先・端末を特定する情報は**入っていません**
- 送るかどうかは、あなたのタップで決まります(名前もメールも不要の匿名フォームが開くだけ)

## 作った人について

エンジニアです。夜職の当事者ではありません。

相場の情報は、お店と求人サイトの側にはたくさんあるのに、はたらく人の側からは「自分の適正」が見えない——値段を決める材料が、片側にしかない。この不公平が、ずっと気になっていました。

値段を**決められる**側から、**確かめる**側へ。そのための小さな道具です。

広告も課金もありません。データを売ることもありません。集まったデータは、丸めた統計としてこのツールに反映される以外の使いみちを持ちません。
